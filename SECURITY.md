# Security Policy

## Supported versions

This repository is currently maintained on the latest mainline version in this workspace.

## Reporting a vulnerability

Please do not open a public issue with exploit details, secrets, or proof-of-concept payloads.

Preferred disclosure paths:

1. Use the hosting platform's private security advisory workflow if available.
2. Contact the maintainers privately if a contact address has been published alongside this repository.
3. If neither option exists, open a minimal public issue that states only that a security concern exists, and avoid technical details until a private channel is arranged.

## What to include

- A short summary of the issue
- Affected file or module, if known
- Reproduction steps, if safe to share privately
- Any logs or screenshots with secrets removed

## Not a security report

Use normal issues or pull requests for:

- Usability bugs
- Translation fixes
- General refactors
- Feature requests
