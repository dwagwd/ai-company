# Code of Conduct

This project is intended to stay respectful, collaborative, and practical.

## Expected behavior

- Be respectful and constructive.
- Keep feedback focused on the code, design, and behavior.
- Prefer specific technical criticism over personal commentary.
- Be careful with private data, secrets, and user content.

## Unacceptable behavior

- Harassment, threats, or discriminatory language
- Deliberate disruption of discussions or pull requests
- Sharing secrets, credentials, or private user data
- Repeatedly ignoring maintainer guidance on safety or scope

## Enforcement

Maintainers may remove content, lock threads, or block participation when needed to keep the project usable for everyone.

## Reporting concerns

Use `SECURITY.md` for vulnerability reports. For other conduct concerns, open a private issue or contact the maintainers through the project’s published support channel.
